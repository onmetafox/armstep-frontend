## Getting Started

### Prerequisites

Need to have installed before they can use your project.

- Node.js (18.0.0)
- yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/armansw/armstep-frontend.git
```

2. Install packages
```
yarn install
```
3. Run dev server
```
yarn start
```
